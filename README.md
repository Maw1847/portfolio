# Portfolio

A personal portfolio built with Angular and Nx, following professional software engineering standards.

## Project Vision

This portfolio highlights:
- Engineering rigor and long-term reliability.
- Architecture thinking and maintainability.
- Accessibility by design.
- Security awareness.

## Getting Started

### Development

Run `npm start` to start the development server.

### Build

Run `npm run build` to build the project.

### Lint

Run `npm run lint` to lint the project.

### Test

Run `npm test` to run unit tests.
Run `npm run e2e` to run e2e tests.
